纯净系统安装教程

教程
2019-11-11

安装流程

  1. 下载系统文件
  2. 下载工具：https://vip.tiandashuiye.cn/onekeyghost.com/2018/09/14/OneKeyGhost.exe
  3. 运行工具：
  4. 这步为还原系统重要的一步，请认真选择:
    1. 首先，选择还原分区
    2. 其次，点击打开找到并选择下载的系统文件
    3. 最后，选择分区，一般默认到第一个分区，大部分是C盘
      这三步选择好之后点击确定：
  5. 此时程序会对镜像文件进行完整性的检查，如果该镜像文件非常完整，可以正常操作安装系统的话，会进行该提示：
  6. 直接点击是进行还原，这时候电脑会黑屏并自动进行重启，这个时候别慌
  7. 待重启时，开机启动项多出Onekey Ghost，默认自动选择Onekey Ghost进入安装过程（如未自动选择，则需手动选择）：
  8. 启动后会进入到一下界面，这是在执行还原系统的操作，不需要任何操作请等待蓝色进度条走完：
  9. 蓝条操作完成后，电脑会自动重启，继续执行系统驱动的安装、系统配置和激活过程（全程自动，无需操作）
  10. 待以上自动化操作完成后，电脑会再次自动重启，这次重启后会进入全新系统的桌面，则说明系统安装成功，请尽情使用新系统吧！

系统只预装了3~5个常用程序：

  1. WINRAR 5.40（正版KEY注册）
  2. 搜狗拼音和QQ五笔输入法（支持卸载）
  3. QQ浏览器（支持卸载）
  4. 由于为了让大家更了解系统的安全性，所以自带了QQ电脑管家，支持卸载

温馨提示（纯净版WIN10系统）：

系统关闭了Windows Update 自动更新，如需要自动更新需自行打开，方法如下：

  1. 按“WIN+R”键，打开运行窗口
  2. 运行窗口输入services.msc 回车即可
  3. 在弹出的服务窗口找到 Windows Update 选择启动。

WIN10激活密钥 神key

激活方法：

运行命令提示符(管理员)
系统安装完毕后，首先以管理员身份打开CMD命令行窗口，按下Win+X，选择命令提示符(管理员)。

输入执行神key
在命令提示符界面依次输入以下内容（可复制粘贴）：

  • win10专业版用户请依次输入：
    slmgr /ipk W269N-WFGWX-YVC9B-4J6C9-T83GX
slmgr /skms kms.xspace.in
slmgr /ato
  • win10企业版用户请依次输入：
    slmgr /ipk NPPR9-FWDCX-D2C8J-H872K-2YT43
slmgr /skms kms.xspace.in
slmgr /ato
